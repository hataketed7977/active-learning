- 标准化参数
	- 不会随着JVM变化而变化
	- 以 `-` 开头，如：-version、-jar
	- 使用 `java -help` 查看
	-
- 非标准化参数
	- 可能会随着JVM变化而变化，变化较小
	- 以 `-X` 开头，如：-Xmx、-Xms、-Xmn、-Xss
	- 使用 `java -X` 查看
- 不稳定参数
	- 会导致JVM不稳定，慎用
	- 也是非标准的参数，主要用于JVM调优与Debug
	- 以 `-XX` 开头，如：-XX：+UseG1GC
	- 不稳定参数分为三类：
		- 性能参数：用于JVM的性能调优和内存分配控制，如内存大小的设置；
		- 行为参数：用于改变JVM的基础行为，如GC的方式和算法的选择；
		- 调试参数：用于监控、打印、输出jvm的信息；
